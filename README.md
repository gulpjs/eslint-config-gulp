# eslint-config-gulp

[![NPM version][npm-image]][npm-url] [![Downloads][downloads-image]][npm-url]

Sharable eslint config for gulp projects.

## Usage

```js
// eslint.config.js
import gulpConfig from "eslint-config-gulp";

export default [
  ...gulpConfig,
];
```
