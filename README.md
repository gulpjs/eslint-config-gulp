# eslint-config-gulp

[![NPM version][npm-image]][npm-url] [![Downloads][downloads-image]][npm-url] [![Gitter chat][gitter-image]][gitter-url]

Sharable eslint config for gulp projects

## Usage

```json
// .eslintrc
{
  "extends": "gulp"
}
```
